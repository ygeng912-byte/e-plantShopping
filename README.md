# e-PlantShopping Application

## Project Overview
This is a React-based e-commerce application designed for an online plant nursery. The application allows users to browse various categories of plants, add them to a shopping cart, and manage their orders. It demonstrates proficiency in React, Redux Toolkit for state management, and client-side routing.

## Key Features
* **Product Catalog:** Displays plants categorized by type (Air Purifying, Aromatic, Insect Repellent) with images and descriptions.
* **Shopping Cart:** Features a fully functional cart where users can:
    * Add items from the product list.
    * Increment or decrement item quantities.
    * Remove items completely.
    * View real-time total costs.
* **State Management:** Utilizes **Redux Toolkit** to manage the global state of the cart across different components.
* **Responsive Design:** Optimized for a seamless user experience.

## Technologies Used
* **React:** Frontend library for building the user interface.
* **Redux Toolkit:** For efficient state management.
* **Vite:** Build tool for faster development.
* **GitHub Pages:** Used for deployment.

## How to Run Locally
1.  Clone the repository:
    ```bash
    git clone [https://github.com/ygeng912-byte/e-plantShopping.git](https://github.com/ygeng912-byte/e-plantShopping.git)
    ```
2.  Navigate to the project directory:
    ```bash
    cd e-plantShopping
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```
